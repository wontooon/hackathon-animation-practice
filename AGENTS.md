# Agent Instructions

## Project Scope

- This is a small static web animation practice project using plain HTML, CSS, and JavaScript.
- Keep changes simple and beginner-friendly unless the task explicitly asks for a larger redesign or framework migration.
- Do not add a build tool, package manager, framework, or external dependency unless it is clearly required by the request.

## File Roles

- `index.html` contains the page structure and copy.
- `style.css` contains layout, visual styling, and `@keyframes` animations.
- `script.js` contains small interaction logic, including animation triggers.

## Local Run

- Open `index.html` directly in a browser.
- There is no install step and no dev server requirement for the current project.

## Change Guidance

- Prefer semantic HTML, readable class names, and straightforward CSS.
- Keep animation names and timing easy to understand for practice use.
- Avoid unrelated cleanup or formatting churn.
- Preserve existing user work and do not revert changes unless explicitly asked.

## Verification

- After visual or interaction changes, open `index.html` in a browser and confirm the animation still works.
- For code-only checks, inspect the changed files for obvious syntax issues and run `git diff --check` when the repository is available.
