# hackathon-animation-practice

A small practice project for learning web animations with plain HTML, CSS, and JavaScript.

## Files

- `index.html` — page structure
- `style.css` — styling and `@keyframes` animations
- `script.js` — triggers the animation on button click

## How to run

Open `index.html` in any browser, then click **Play animation** to watch the box bounce.

## Practice ideas

- Tweak the `@keyframes bounce` values in `style.css`
- Add new animations (rotate, fade, color shift)
- Trigger animations on hover, scroll, or page load
